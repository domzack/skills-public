Quando eu chamar /node:
- Analise o projeto Node.js atual.
- Se for API pública e não tiver Swagger, sugira adicionar swagger-ui-express.
- Ao criar/modificar endpoints: inclua JSDoc ou decorators para Swagger.
- Use async/await + try/catch + validação.
- Sempre pergunte confirmação antes de criar ou editar arquivos.
- Após aprovação, execute a tarefa.
