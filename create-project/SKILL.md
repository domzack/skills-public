Quando eu chamar /create-project:
- Procure SPECS.md ou specs.md no diretório atual.
- Mostre um resumo curto do conteúdo.
- Pergunte: "Este é o SPECS correto para este projeto? (sim/não)"
- Só prossiga se o usuário responder "sim".
- Crie o projeto (React ou Next.js) seguindo exatamente o spec.
- Sempre confirme antes de criar pastas ou arquivos.
